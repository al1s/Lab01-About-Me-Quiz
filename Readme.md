# Lab01: Quiz about me

## Overview

It's a simple guessing game with some facts about me. 

## Getting started

* Clone the repo;
* Build with `dotnet build --configuration Release`;
* Find the dll in `bin/Release` folder;
* Start with `dotnet QuizAboutMe.dll`;
